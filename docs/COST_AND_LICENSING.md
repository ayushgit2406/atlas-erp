# Cost and Licensing

Atlas ERP is designed to be built and run with **zero licensing cost**.

## Core technologies

| Component | License | Cost |
|---------|--------|------|
| Apache Kafka | Apache 2.0 | $0 |
| Apache Avro | Apache 2.0 | $0 |
| PostgreSQL | PostgreSQL License | $0 |
| Redis (self-hosted) | BSD/RSAL | $0 |
| Keycloak | Apache 2.0 | $0 |
| .NET 8 | MIT | $0 |
| Node.js / TypeScript | MIT / Apache 2.0 | $0 |
| NestJS | MIT | $0 |
| Python / FastAPI | MIT | $0 |
| Docker Engine / Compose | Apache 2.0 | $0 |
| pnpm | MIT | $0 |
| Poetry | MIT | $0 |

## Notes
- All infrastructure is **self-hosted**.
- No SaaS subscriptions or per-request pricing.
- Docker Desktop licensing is not required for Linux and is free for personal/portfolio use.

Total licensing cost: **$0**.
